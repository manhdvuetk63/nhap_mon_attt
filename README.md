# Nhập môn an toàn thông tin ⚠

## Cách cài đặt

- Cài đặt npm 
- Cài đặt các dependency, chạy lệnh
  
  ```
  npm install
  ```
- Khởi chạy
  ```
  npm start
  ```
- Chương trình chạy ở http://localhost:3000/
